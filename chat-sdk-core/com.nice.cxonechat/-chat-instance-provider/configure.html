configure

fun configure(context: Context, actions: ChatInstanceProvider.ConfigurationScope.() -> Unit)

Update the configuration of chat.

  1. Stops any current chat. This will result in discarding any stored Authorization or UserName. Any such details must be provided in the configuration block once again.

  2. Executes the configuration actions block.

  3. Restarts chat.

Parameters

context

Application context for resource access.

actions

Actions to reconfigure the chat.
